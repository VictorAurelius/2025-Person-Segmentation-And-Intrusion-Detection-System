## Hướng Dẫn Thực Hiện

### Bước 1: Thu thập tài liệu đã sử dụng

Kiểm tra tất cả tài liệu được tham khảo trong:
- `documentation/01-theory-foundation/references.md`
- Code comments
- Các file trong `knowledge-base/`

### Bước 2: Phân loại tài liệu

Phân loại theo các nhóm:
- Sách giáo trình
- Papers
- Documentation
- Datasets
- Tutorials
- GitHub repos
- Tools và libraries

### Bước 3: Format theo chuẩn

Sử dụng format IEEE hoặc APA:
- Đánh số liên tục [1], [2], ...
- Ghi rõ tác giả, năm, tên tài liệu
- Ghi rõ nguồn (sách, conference, journal, URL)

### Bước 4: Sắp xếp

- Sắp xếp theo thứ tự xuất hiện trong báo cáo
- Hoặc sắp xếp theo alphabet (tùy yêu cầu giảng viên)

### Bước 5: Tạo file output

Tạo file `documentation/report/04-references.md` với toàn bộ nội dung.

---

## Yêu Cầu Format

1. **Đánh số liên tục**: [1], [2], [3], ...
2. **Thụt lề**: Dòng thứ 2 trở đi thụt vào 4 spaces
3. **Đầy đủ thông tin**: Tác giả, năm, tiêu đề, nguồn
4. **URL**: Ghi rõ đường dẫn truy cập cho tài liệu online
5. **Ngôn ngữ**: Giữ nguyên tên gốc (tiếng Anh) cho tài liệu quốc tế

---

## Lưu Ý Quan Trọng

1. **Trích dẫn chính xác**: Đảm bảo tất cả tài liệu được trích dẫn trong báo cáo đều xuất hiện trong danh sách
2. **Không trích dẫn thừa**: Chỉ liệt kê tài liệu thực sự sử dụng
3. **Kiểm tra URL**: Đảm bảo các link còn hoạt động
4. **Chuẩn format**: Nhất quán theo IEEE hoặc APA (chọn 1)
5. **Năm xuất bản**: Ghi năm chính xác, sử dụng năm truy cập cho tài liệu online
