## 2. Gradient-based Operators

### A. Sobel Operator

```python
# Sobel kernels
Gx = [[-1, 0, 1],
      [-2, 0, 2],
      [-1, 0, 1]]

Gy = [[-1, -2, -1],
      [ 0,  0,  0],
      [ 1,  2,  1]]

# Implementation
grad_x = cv2.Sobel(gray, cv2.CV_64F, 1, 0, ksize=3)
grad_y = cv2.Sobel(gray, cv2.CV_64F, 0, 1, ksize=3)
magnitude = np.sqrt(grad_x**2 + grad_y**2)
```

## 3. Canny Edge Detection

### 5 Bước Của Canny

1. **Gaussian blur** - Giảm nhiễu
2. **Gradient calculation** - Tính gradient
3. **Non-maximum suppression** - Làm mỏng edges
4. **Double thresholding** - Phân loại edges
5. **Edge tracking by hysteresis** - Nối edges

### Implementation

```python
edges = cv2.Canny(gray, low_threshold, high_threshold)
```

**Tham số:**
- `low_threshold`: Ngưỡng dưới
- `high_threshold`: Ngưỡng trên
- Ratio khuyến nghị: 1:2 hoặc 1:3

## 4. Parameter Tuning

# Nhiều edges
edges = cv2.Canny(gray, 30, 100)

# Cân bằng
edges = cv2.Canny(gray, 50, 150)

edges = cv2.Canny(gray, 100, 200)
